How to parallelize with bash


April 9, 2021



I'm trying to write this script that'll run several bq-runner jobs for me and I realized that I never end up using bash regularly so even when I do end up learning something, I end up forgetting it by the time I end up needing it again (which is usually several months later).

Sad!

So here's my attempt to formalize that knowledge for myself. Maybe I'll even include a link to a gist

            
                
                
my script for running daily backfills

                
#!/bin/bash
# Perform daily backfills
set -eu

project=$1
dataset=$2
table=$3

declare -a dates=(
'2020-09-29'
'2020-09-30'
'2020-10-01')

bqtask(){
        sleep 0.2; 
        echo "*********** running backfill for ${i} ***********"
        bq-runner RunYaml  --partition ${i} --target ${project}.${dataset}.${table}_YYYYMMDD  --template-path bigquery --local-scheduler --overwrite
        echo "*********** backfill complete for ${i} partition  **************"
        }
N=4
(
for date in "${dates[@]}"; do
        ((i=i%N)); ((i++==0)) && wait
        bqtask "$date" &
done
)
